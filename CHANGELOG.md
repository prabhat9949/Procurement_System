# Changelog

All notable changes to the Enterprise Procurement System are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and the project uses semantic versioning once releases are established.

## [Unreleased]

### Added

- Initial repository structure for the Spring Boot backend, React frontend,
  MySQL database, documentation, Postman collections, scripts, assets, and
  GitHub configuration.
- Project documentation describing the procurement lifecycle, architecture,
  modules, roles, setup, workflow, and development roadmap.
- Contribution guidelines, project license, and technology-specific ignore rules.

### Planned

- Authentication and JWT-based RBAC.
- Master data and procurement request modules.
- Approval workflow and budget validation.
- RFQ, quotation, purchase order, GRN, inventory, invoice, and payment modules.

## Release Guidelines

Before each release, move completed items from `Unreleased` into a dated,
versioned section and link the release to its repository tag.
