# Enterprise Procurement System

## Overview

The **Enterprise Procurement System (EPS)** is a comprehensive, enterprise-grade procurement management platform designed to automate and streamline the complete procurement lifecycle within an organization. It provides a centralized solution for managing purchase requests, approvals, vendor interactions, purchase orders, inventory, invoices, payments, and reporting through a secure, role-based workflow.

The system enables employees to raise purchase requisitions, which are automatically routed through configurable multi-level approval processes involving department managers, finance teams, and procurement managers. Once approved, the procurement team can initiate vendor selection through RFQs, generate purchase orders, manage deliveries, and update inventory using Goods Receipt Notes (GRNs). Finance users complete the procurement cycle by performing three-way matching between Purchase Orders, Goods Receipts, and Vendor Invoices before processing payments.

Designed using a modular architecture, the application ensures transparency, accountability, and operational efficiency by integrating role-based access control, audit logging, real-time notifications, budget validation, workflow automation, and analytical dashboards into a single platform.
